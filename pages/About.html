This website is a project created by ICT students for their Research Project.
This project aims to help school administrators (in this case, the Guidance Council)
to track and monitor students' infractions faster. The data that is presented within
the website is stored locally into the device of the user and cannot be accessed by other people.
The researchers hope that this website will be useful to school administrators. Maligayang Araw!

Please visit the Contact Page to contact the researchers and get assistance.
